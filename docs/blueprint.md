# **App Name**: MotoLog

## Core Features:

- Motorcycle Profile Management: Users can add, view, and edit detailed profiles for multiple motorcycles, including make, model, year, and specific identification numbers.
- Trip & Usage Logging: Record details for each ride, including date, distance (kilometers), and maximum speed achieved. This data automatically updates the total mileage.
- Maintenance Scheduling & Tracking: Log past service records, schedule upcoming maintenance tasks, and receive smart reminders, with special alerts for critical items like oil level.
- Fuel & Expense Management: Track all expenses, including fuel purchases, repairs, parts, and other related costs, with categorization options for financial oversight.
- Dynamic Dashboard Overview: A centralized and personalized dashboard presenting critical motorcycle statistics, such as current mileage, estimated oil life, and a feed of recent activities.
- Predictive Oil Life Indicator: Automatically calculates and displays the estimated remaining oil life percentage based on recorded usage data and predefined thresholds, providing proactive maintenance insights.
- AI-Powered Trip Description Tool: An AI tool that suggests SEO-friendly titles and semantically rich descriptions for recorded trips, enhancing content discoverability and organization.

## Style Guidelines:

- The dominant color scheme is dark mode, with a very dark background color (#121212) to reduce eye strain and provide a sleek, modern look. This dark backdrop allows vibrant UI elements to stand out with clarity.
- Primary accent color: A vibrant, energetic blue(#0022EE). This hue is used for interactive elements, key data highlights (like current mileage and oil life), and calls to action, evoking a sense of 'control and vitality'.
- Secondary accent color: A vivid orange (#F27D00) is strategically used for alerts, notifications, and secondary actions that require immediate attention or provide complementary visual information. This color ensures strong contrast and visibility against the dark background.
- All text uses 'Inter', a versatile sans-serif font known for its clean lines and excellent readability across various sizes. This choice reflects a modern, tech-forward, and objective aesthetic, suitable for numerical data and informative content.
- Utilize modern, minimalist line icons throughout the application. These icons should be easily discernible against the dark background, with a consistent style that enhances navigability and visual appeal (e.g., stylized motorcycle icon, wrench for maintenance, ledger for expenses, person for profile).
- The application features an intuitive bottom navigation bar for quick access to main sections (Dashboard, Maintenance, Expenses, Profile). Information is presented in a card-based layout for clear data segmentation, and a floating action button (+) provides convenient shortcuts for logging new data entries.
- Implement subtle, fluid animations for state changes, navigation transitions, and data updates. These animations should be fast and responsive, providing visual feedback to user interactions without feeling distracting or slow, enhancing the overall user experience.
